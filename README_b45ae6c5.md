# ZenPop 解压乐园

一款零依赖的移动端解压小游戏合集，单文件即可运行。集合 5 种经典解压玩法，治愈系配色搭配流畅动画与触觉反馈，帮你随时随地释放压力。

## 游戏模式

|模式|说明|
|-|-|
|🫧 泡泡|点击挤破满屏气泡，粒子爆裂 + 触感反馈，挤完可一键重置|
|🎨 涂色|曼陀罗几何图案涂色，12 色调色盘，支持撤销，完成时有庆祝动画|
|🔨 破坏|疯狂点击破坏花瓶、玻璃杯等物品，碎裂粒子动画 + 连击手感|
|🌊 沙盘|Canvas 粒子流沙模拟，手指滑动产生粒子流，支持金沙/彩虹/星空三种沙色|
|🧘 呼吸|4-7-8 呼吸法引导，跟随圆形动画调整呼吸节奏，背景色随状态变化|

## 特性

* **移动端优先** — 适配手机屏幕与安全区域，触摸事件优化
* **零依赖** — 单个 HTML 文件，无需任何框架或外部资源
* **音效反馈** — Web Audio API 实时生成音效，右上角可开关
* **触觉反馈** — 支持设备振动 API，交互更有手感
* **PWA 支持** — 可添加到手机主屏幕，全屏使用
* **解压计时** — 顶部全局计时器记录你的解压时长
